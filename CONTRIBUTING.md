# Contributing to SerapeumAI

Thank you for your interest in contributing to SerapeumAI! This project aims to bring AI-powered architectural intelligence to every professional's desk.

## How to Contribute

### 1. Reporting Bugs
- Use the **Bug Report** template in GitHub Issues.
- Provide as much detail as possible, including logs from `.serapeum/logs/`.

### 2. Suggesting Enhancements
- Use the **Feature Request** template.
- Explain why the feature would be useful for the AECO industry.

### 3. Code Contributions
- **Fork** the repository.
- Create a new branch: `git checkout -b feat/your-feature-name`.
- Commit your changes: `git commit -m 'feat: add some amazing feature'`.
- **Push** to the branch: `git push origin feat/your-feature-name`.
- Open a **Pull Request**.

### 4. Development Environment
- Install **Python 3.11**.
- Install **Node.js** (for LM Studio CLI).
- Run `Setup.bat` to install system dependencies.
- Run `pip install -r requirements.txt`.

## Pull Request Guidelines
- Ensure your code is well-documented.
- Run existing tests using `pytest` before submitting.
- Be respectful and constructive in your communication.
