# Security Policy

## Supported Versions

Currently, only the latest version of SerapeumAI is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| < 0.2.x | :x:                |

## Reporting a Vulnerability

We take the security of our users' data seriously. If you find a security vulnerability, please do NOT open a public issue. Instead, please report it privately:

- **Email**: [security@serapeum.ai] (Placeholder — update with your real email)
- **GitHub**: You can also use the private vulnerability reporting feature on GitHub if enabled.

Once a report is received, we will:
1. Acknowledge the receipt within 48 hours.
2. Work on a fix and provide a timeline.
3. Release a security advisory once the fix is deployed.
